Android ID ANE
==============

Enables Adobe AIR to access ANDROID_ID. That's it.

app.xml
-------

```
<extensions>
    <extensionID>com.mesmotronic.ane.androidid</extensionID>
</extensions>
``` 

ActionScript
------------

```actionscript
import com.mesmotronic.ane.AndroidID;

trace(AndroidID.ANDROID_ID);
```
